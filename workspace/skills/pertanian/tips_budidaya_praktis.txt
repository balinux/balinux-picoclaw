TIPS PRAKTIS BUDIDAYA TANAMAN 2026 🌾🌽🌶🍅

PERTANIAN MODERN UNTUK HASIL MAKSIMAL

1. PADI SISTEM JAJAR LEGOWO 2:1
 ✓ Atur jarak tanam 20cm x 10cm x 40cm
 ✓ Dua baris tanaman diselingi satu baris kosong
 ✓ Meningkatkan cahaya matahari dan sirkulasi udara
 ✓ Hasil panen meningkat hingga 20-30%
 ✓ Gunakan benih unggul usia 18-25 hari

2. JAGUNG HIBRIDA
 ✓ Pilih varietas unggul sesuai lahan
 ✓ Jarak tanam ideal: 70-75cm x 20cm
 ✓ Gunakan 1 biji per lubang tanam
 ✓ Beri pupuk organik saat tanam
 ✓ Panen usia 80-90 hari tergantung varietas

3. CABAI RAWIT/HIBRIDA
 ✓ Gunakan sistem hidroponik wick system
 ✓ Media tanam: cocopeat atau rockwool
 ✓ Butuh sinar matahari 6-8 jam/hari
 ✓ Kontrol pH larutan 5.5-6.5
 ✓ Panen mulai umur 2-3 bulan

4. TOMAT CHERRY
 ✓ Gunakan sistem hidroponik NFT atau dutch bucket
 ✓ Semai pada rockwool, beri nutrisi lengkap
 ✓ Lakukan pemangkasan tunas air rutin
 ✓ Perlukan pengikatan batang saat tumbuh
 ✓ Panen saat buah berwarna merah cerah
